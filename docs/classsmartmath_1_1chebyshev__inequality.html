Detailed Description

The chebyshev_inequality class contains methods and utils to deal with the chebyshev inequality, which guarantees that, for a wide class of probability distributions, "nearly all" values are close to the mean—the precise statement being that no more than 1/k^2 of the distribution's values can be more than k standard deviations away from the mean (or equivalently, at least 1−1/k2 of the distribution's values are within k standard deviations of the mean)
