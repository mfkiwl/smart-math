template<class T>
class smartmath::integrator::base_integrationwevent< T >

The base_integrationwevent class is a template abstract class. Any integrator handling events added to the toolbox needs to inherit from it and implement the method integrate with events.

The base_integrationwevent class is a template abstract class. Any integrator handling events added to the toolbox needs to inherit from it and implement the method that integrates while dealing with events
