twobody constructor

The constructor initialize the problem parameters and scaling factors to the value supplied by the user. The model parameters (10 in total) are supplied in order as:

  • param 1-3: thrust value alng each direction
  • param 4:
  • param 5-6: atmospheric parameters (exponential atmospheric model is considered)
  • param 7: produc between aerodynamic coefficient and surface area
  • param 8-10: unknown acceleration (3 components) Default values for scaling factors are 1 and parameters is a vector of zero values or zero polynomials.
